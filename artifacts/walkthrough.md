# Insta-Lite Walkthrough

Insta-Lite는 워크숍용으로 개발된 경량 인스타그램 클론입니다. FastAPI, SQLite, Vanilla JS를 사용하여 핵심 기능을 구현했습니다.

## 🚀 실행 방법

프로젝트 루트 디렉토리에서 아래 명령어를 실행하여 서버를 시작합니다.

```bash
# 가상환경 활성화 (필요 시)
source .venv/bin/activate

# 서버 실행 (개발 모드)
uvicorn app:app --host 0.0.0.0 --port 8000 --reload
```

서버가 시작되면 브라우저에서 [http://localhost:8000](http://localhost:8000)으로 접속합니다.

## ✨ 주요 기능 확인

### 1. 회원가입 및 로그인
- 상단 내비게이션의 **회원가입** 버튼을 클릭하여 계정을 생성합니다.
- 가입 후 **로그인** 페이지로 이동하여 이메일과 비밀번호로 로그인합니다.
- 로그인 성공 시 메인 피드 화면으로 이동하며, 상단에 **로그아웃** 버튼이 표시됩니다.

### 2. 게시글 작성
- 로그인 상태에서 메인 화면 상단에 **새 게시물 작성** 카드가 나타납니다.
- 텍스트를 입력하고 이미지를 선택하여 **게시하기** 버튼을 누릅니다.
- 잠시 후 피드에 내 글이 최상단에 추가되는지 확인합니다.

### 3. 상호작용 (좋아요/댓글)
- 게시물의 하트 아이콘(❤️)을 눌러 좋아요를 표시하거나 취소할 수 있습니다.
- 댓글 아이콘(💬)을 누르면 댓글 목록과 작성 폼이 나타납니다.

### 4. 반응형 디자인 확인
- 브라우저 창 크기를 조절하거나 개발자 도구(F12)의 모바일 모드를 켜서 레이아웃이 자연스럽게 변경되는지 확인합니다.
- 모바일에서는 1열, 데스크탑에서는 다열 그리드로 표시됩니다.

## 🧪 테스트 실행

API 엔드포인트가 정상 작동하는지 자동화된 테스트를 실행할 수 있습니다.

```bash
python test_app.py
```

"All tests passed!" 메시지가 출력되면 모든 기능이 정상입니다.

## 📂 프로젝트 구조

- `app.py`: FastAPI 메인 애플리케이션 및 API 라우트
- `database.py`: DB 연결 및 초기화 로직
- `auth.py`: 비밀번호 해시 및 JWT 토큰 처리
- `templates/`: HTML 템플릿 (Jinja2)
- `static/`: CSS, JS, 정적 리소스
- `uploads/`: 업로드된 이미지 저장소
- `insta_lite.db`: SQLite 데이터베이스 파일
